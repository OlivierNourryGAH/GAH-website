Accessibility statement

Accessibility Statement for Games Accessibility Hub's corporate website

This is an accessibility statement from Games Accessibility Hub.

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. Games Accessibility Hub's corporate site is partially conformant with WCAG 2.2 level AA. Partially conformant means that some parts of the content do not fully conform to the accessibility standard.

Compatibility with browsers and assistive technology

Games Accessibility Hub's corporate site is designed to be compatible with a wide array of assistive technologies, devices, operating systems, and browsers. We tested different combinations, and found no significant accessibility issue with our own code and content.

Technical specifications

Accessibility of Games Accessibility Hub's corporate site relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • WAI-ARIA
  • CSS
  • JavaScript

These technologies are relied upon for conformance with the accessibility standards used.

Limitations and alternatives

Despite our best efforts to ensure accessibility of Games Accessibility Hub's corporate site, there may be some limitations. Below is a description of known limitations, and potential solutions. Please contact us if you observe an issue not listed below.

The known limitations for Games Accessibility Hub's corporate site relate to the Playtesters recruitment form, which is an embedded Google Form:

  1. The menu displayed when the "More" button is activated has minor issues of compatibility with assistive technologies
  2. The contrast for the "More" button is insufficient
  3. The visual indication of the focus position is missing on the "Clear form" link, and not visible enough on the "Submit" button
  4. On certain Android smartphones, the recruitment form doesn't fit within the width of the screen. While still usable, it may cause incomfort in some situations

Unfortunately we have no control over the design nor the code embedded in this form. It is a temporary solution, while we are working on a custom, more accessible form. The "More" button and its menu can be used with assistive technologies, but with a suboptimal experience for users of screen readers, in particular.

These are two of the three links you can find in this menu: Google's Terms of service and Google's Privacy policy. Unfortunately we can't provide a link to the "Report abuse" page, since it's generated when the form is displayed, and changes every time. Hopefully, there is no need for this feature, since this form does not fall under any of the potential causes for abuse report.

Assessment approach

We assessed the accessibility of Games Accessibility Hub's corporate site by self-evaluation.
